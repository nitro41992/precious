# Reset Requirements

This fresh app supersedes the older mobile implementation for now.

## In scope

- Android share sheet intake for text and links.
- Local durable capture storage on device.
- Processing notification immediately after share.
- Completion notification when background processing finishes.
- Tapping completion opens the capture in Quick Edit.
- Quick Edit supports title and note.
- Home shows a minimal searchable list of precious captures.
- Visual design is flat, quiet, and almost white.

## Out of scope

- Auth.
- Supabase.
- Collections.
- Agenda.
- Library.
- Web dashboard.
- Visual parity with the old concept.
- AI model calls.

## Product shape

The first proof is not a full knowledge system. It is a trusted pocket for things worth keeping. Capture must be instant; processing can be small and visible; editing must be one tap away from the completion notification.
